# `figures/`

Output plots generated by scripts. All figures are saved as `.png` at 150 dpi.

| File | Generated by | Description |
|---|---|---|
| `data_overview.png` | `explore_data.py` | Scatter plots of Gaussian and Moons datasets, plus a sample digit image |
| `decision_boundaries.png` | `plot_decision_boundary.py` | 2×2 grid comparing Softmax vs NN boundaries on Gaussian and Moons |
| `capacity_ablation.png` | `capacity_ablation.py` | NN decision boundaries on Moons across hidden widths {2, 8, 32} |
| `digits_training_dynamics.png` | `compare_digits.py` | Loss and accuracy curves for Softmax vs NN on digits benchmark |
| `optimizer_study.png` | `optimizer_study.py` | Loss and accuracy curves comparing SGD, Momentum, and Adam |
| `reliability_diagram.png` | `plot_reliability.py` | Reliability (calibration) curves for Softmax and NN on digits test set |
| `confidence_hist.png` | `plot_reliability.py` | Confidence histograms split by correct/incorrect predictions |
| `entropy_boxplot.png` | `plot_reliability.py` | Predictive entropy boxplots split by correct/incorrect predictions |
